Rent Management System

Description

This is a simple Rent Management System developed using HTML, CSS, and JavaScript. It helps landlords manage houses, tenants, rent payments, reports, and payment receipts. The system stores data using the browser's Local Storage.

Features

- Admin Login
- Dashboard
- House Management
- Tenant Registration
- Rent Payment
- Reports
- Payment Receipts

Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

How to Run

1. Open the project folder.
2. Open login.html in a web browser.
3. Log in and use the dashboard to manage houses, tenants, payments, and reports.
